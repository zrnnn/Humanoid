# Humanoid Bavaria - Home of Humanoids

**Note: This is a test website created for testing AI agentic capabilities.**

This repository contains the source code for the Humanoid Bavaria website, a competency network for humanoid robotics in Bavaria.

## 🚀 Live Demo

The website is automatically deployed to GitHub Pages:
[https://zrnnn.github.io/Humanoid/](https://zrnnn.github.io/Humanoid/)

## 🛠 Tech Stack

- **React** (TypeScript)
- **Vite**
- **Tailwind CSS**
- **Framer Motion** (for animations)
- **Three.js / React Three Fiber** (for 3D elements)

## 📂 Project Structure

- `src/components/`: Modular React components.
- `src/assets/`: Static assets and images.
- `.github/workflows/`: Automated deployment via GitHub Actions.
